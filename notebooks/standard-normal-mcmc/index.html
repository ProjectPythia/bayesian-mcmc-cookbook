A MCMC implementation for a standard normal distribution¶

Target Distribution (Posterior)¶

In notebook, the goal is to generate samples from the standard normal distribution (a normal distribution centered around 0 with standard deviation of 1):

p(x)=12πexp⁡(−x22)p(x) = \frac{1}{\sqrt{2\pi}}\exp\left(-\frac{x^2}{2}\right)p(x)=2π​1​exp(−2x2​)

Since there is no data (I’m not conditioning on any observations for this simple example), the posterior is simply the target distribution p(x)p(x)p(x), the standard normal density.

MCMC steps¶

  1. Proposal Step:

    • Propose a new point x∗x^*x∗ from a normal distribution centered at the current xxx:
      x∗∼N(x(t−1),σ2)x^* \sim \mathcal{N}(x^{(t-1)}, \sigma^2)x∗∼N(x(t−1),σ2)
      (3)
    • This is a random walk Metropolis proposal: q(x∗∣x(t−1))=N(x(t−1),σ2)q(x^* | x^{(t-1)}) = \mathcal{N}(x^{(t-1)}, \sigma^2)q(x∗∣x(t−1))=N(x(t−1),σ2).

  2. Acceptance Ratio:

    • Since the proposal distribution is symmetric, the qqq terms cancel out in the general M-H acceptance ratio:
      α=min⁡(1,p(x∗)p(x(t−1)))\alpha = \min\left(1, \frac{p(x^*)}{p(x^{(t-1)})}\right)α=min(1,p(x(t−1))p(x∗)​)
      (4)
    • Plugging in the normal density:
      α=min⁡(1,exp⁡(−(x∗)22)exp⁡(−(x(t−1))22))\alpha = \min\left(1, \frac{\exp\left(-\frac{(x^*)^2}{2}\right)}{\exp\left(-\frac{(x^{(t-1)})^2}{2}\right)}\right)α=min⎝⎛​1,exp(−2(x(t−1))2​)exp(−2(x∗)2​)​⎠⎞​

    • Accept x∗x^*x∗ with probability α\alphaα. Otherwise, keep the current value.
import numpy as np
import matplotlib.pyplot as plt
from numba import njit
def MHSTDnormal(N, X0, sigmaSq):
    x = np.empty(N)
    x[0] = X0
    for i in range(1, N):
        xhat = np.random.normal(x[i-1], np.sqrt(sigmaSq))
        accept_prob = np.exp(-0.5 * (xhat**2 - x[i-1]**2))
        if np.random.uniform() <= accept_prob:
            x[i] = xhat
        else:
            x[i] = x[i-1]
    return x
normalSamples = MHSTDnormal(N = 10000, # number of iterations
                            X0 = 0.3,  # starting value
                            sigmaSq = 10**1 # variance of the standard normal (sd = 1, var = 1^2)
                           )

plt.plot(normalSamples)
plt.title('Metropolis-Hastings samples from Standard Normal')
plt.xlabel('Iteration')
plt.ylabel('Sample Value')
plt.show()

plt.figure(figsize = (8,4))
count, bins, ignored = plt.hist(normalSamples, bins = 50, density = True, alpha = 0.6, color = "skyblue")

x = np.linspace(-4, 4, 1000)
plt.plot(x, 1/np.sqrt(2*np.pi) * np.exp(-x**2/2), 'r', lw=2, label='True Standard Normal')

plt.title('Posterior Distribution')
plt.xlabel('Sample Value')
plt.ylabel('Density')
plt.legend()
plt.show()
